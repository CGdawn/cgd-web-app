# **App Name**: AetherCore

## Core Features:

- AI Virtual Assistant: An AI tool acting as a chatbot to provide quick information about services, project status (for authorized users), and answer FAQs, reflecting the AI-driven solutions offering.
- Secure User Authentication & RBAC: A comprehensive login, registration, and email verification system for various user roles (Super Admin, Admin, Staff, Client, etc.), ensuring role-based access control to different application sections, backed by a scalable database.
- Role-Specific Dashboards: Personalized dashboard views for Admins, Staff, and Clients, displaying relevant information, metrics, and actionable items tailored to their roles within the platform.
- Dynamic Service & Portfolio Showcase: Interactive sections to highlight CG DAWN's diverse range of services (game development, AI solutions, VR/AR, etc.) and a dynamic portfolio to display featured projects.
- Basic Project Tracking: A fundamental project management interface allowing project creation, task assignment, and basic progress tracking, possibly with a Kanban board view, enabling collaboration between staff and clients using a database.
- Staff & Client Profiles Management: Secure administrative areas to manage staff and client information, including roles, contact details, and associated projects, stored in a database.
- Contact & Career Submission Forms: Dedicated and responsive forms for website visitors to send general inquiries or apply for job openings, efficiently capturing essential contact and application details.

## Style Guidelines:

- Primary color: A vibrant, deep violet (#8B3DEE) for a strong futuristic and sophisticated brand presence.
- Background color: A very dark charcoal with a subtle purple undertone (#1A161E) to ensure a true 'dark mode by default' experience that supports the vibrant primary color.
- Accent color: A brilliant, electric blue (#86B6FF) used for interactive elements, highlights, and calls-to-action, providing a neon glow in contrast to the dark theme.
- Headline font: 'Space Grotesk' (sans-serif) for its futuristic, tech-inspired, and bold aesthetic.
- Body font: 'Inter' (sans-serif) for clean readability, modern appearance, and neutrality across all content.
- Sleek, minimalist, and potentially glowing or glassmorphic icons that complement the cyber-tech and gaming-inspired UI/UX, maintaining consistency with the dark theme.
- Predominantly glassmorphism UI elements with clear distinctions between interactive and static components, ensuring a mobile-first responsive design for all interfaces.
- Dashboard layouts will feature modular analytics cards, intuitive sidebar navigation, and dynamic content presentation like Kanban boards and charts.
- Smooth motion transitions between pages and states, subtle scroll animations, dynamic hover effects for interactive elements, and an animated hero section with particle backgrounds to enhance interactivity and a futuristic feel.
